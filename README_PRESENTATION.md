# SPARQ Pitch Deck

## How to View

Simply open `index.html` in your browser:

1. Double-click `index.html` in Finder, OR
2. Right-click → Open With → Your preferred browser, OR
3. Drag `index.html` into an open browser window

## Navigation

- **Next slide**: Click → button, press Right Arrow, or press Spacebar
- **Previous slide**: Click ← button or press Left Arrow
- **Mobile**: Swipe left/right

## Files

- `index.html` - The presentation structure
- `styles.css` - All styling
- `script.js` - Navigation logic

## Customization

Edit `index.html` to change content
Edit `styles.css` to adjust colors, fonts, spacing
